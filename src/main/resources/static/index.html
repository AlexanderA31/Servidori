Compartir Impresoras Locales

Compartir Impresora USB/Local

Comparte una impresora conectada localmente a tu computadora con la red

Instrucciones para Windows:

  1. Descarga el script PowerShell
  2. Haz clic derecho en el archivo descargado y selecciona "Ejecutar con PowerShell"
  3. Si aparece un mensaje de seguridad, selecciona "Abrir" o "Ejecutar de todas formas"
  4. Selecciona la impresora local que deseas compartir
  5. El script compartirá la impresora vía IPP automáticamente

⚠️ Importante: Abre PowerShell como Administrador y ejecuta:
powershell -ExecutionPolicy Bypass -File .\compartir-impresora-windows.ps1

Instrucciones para Linux:

  1. Descarga el script bash
  2. Dale permisos de ejecución: chmod +x share-printer.sh
  3. Ejecuta con sudo: sudo ./share-printer.sh
  4. El script configurará CUPS para compartir la impresora

Configurar Impresora IPP en Cliente

Configura una computadora cliente para usar las impresoras compartidas

URL IPP:

Windows:

  1. Abre "Configuración" → "Dispositivos" → "Impresoras y escáneres"
  2. Haz clic en "Agregar impresora o escáner"
  3. Selecciona "La impresora que deseo no está en la lista"
  4. Selecciona "Seleccionar una impresora compartida por nombre"
  5. Ingresa la URL IPP mostrada arriba
  6. Instala los controladores cuando se solicite

📋 Cómo ejecutar el script:

1. Abre PowerShell como Administrador
Haz clic derecho en el botón de Windows → "Windows PowerShell (Administrador)"

2. Navega a la carpeta de descargas:
cd $env:USERPROFILE\Downloads

3. Ejecuta el script con este comando:
powershell -ExecutionPolicy Bypass -File .\instalar-impresora-[nombre].ps1
(Reemplaza [nombre] con el nombre real del archivo)

Linux:

  1. Abre "Configuración del sistema" → "Impresoras"
  2. Haz clic en "Agregar" o "+"
  3. Selecciona "Impresora de red" → "Impresora IPP"
  4. Ingresa la URL IPP mostrada arriba

O usa este comando:

macOS:

  1. Abre "Preferencias del Sistema" → "Impresoras y Escáneres"
  2. Haz clic en el botón "+" para agregar una impresora
  3. Mantén presionada la tecla "Option" y haz clic en "Avanzado"
  4. Selecciona "Impresora de Internet - IPP"
  5. Ingresa la URL IPP mostrada arriba
  6. Selecciona el controlador apropiado
